Program Magister

Program Magister di Universitas Indonesia adalah program yang ditujukan untuk siswa yang telah lulus dari program sarjana dan ingin melanjutkan studi ke tingkat yang lebih tinggi. Tujuan dari program ini adalah untuk menyediakan siswa dengan keterampilan dan pengetahuan yang diperlukan untuk mengejar karir di bidang tertentu atau melanjutkan studi ke tingkat yang lebih tinggi lagi. Jalur masuk program magister adalah melalui SIMAK UI untuk program pascasarjana.
